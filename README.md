# Platanus

Platan.us toolbelt.

This gem includes various ruby and rails utility classes we use across most of our rails proyects.

## Installation

Add this line to your application's Gemfile:

    gem 'platanus'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install platanus
